# phonepe-digital-payments-python

## Project Overview
This project provides a comprehensive analysis of the PhonePe Pulse dataset, spanning from Q1 2018 to Q2 2021. It explores transaction trends, user demographics, and device usage across various states and districts in India to uncover insights into the adoption of digital payments.

## Objectives
Data Exploration: Load and understand multi-sheet transaction, user, and demographic data.
Transaction Analysis: Evaluate transaction types (e.g., Peer-to-peer, Merchant payments, Bill payments) and their respective growth over time.
Geographic Insights: Identify high-performing and emerging regions based on transaction volume and amount.
User Behavior: Analyze device brand preferences and app usage patterns among registered users.

## Dataset Description
The analysis utilizes several key datasets provided in the phonepe-pulse_raw-data file:
State Transactions & Users: Summary of total transactions, amounts, and registered users per state.
Transaction Split: Breakdown of transactions by category (e.g., Financial Services, Recharges).
Device Data: Insights into the mobile brands used by the PhonePe user base (e.g., Xiaomi, Samsung, Vivo).
District-level Data: Granular transaction and user metrics at the district level.
District Demographics: Population, area, and density information to contextualize payment trends.

## Key Findings & Recommendations
Growth Correlation: Transaction growth shows a strong correlation with demographic density and digital awareness.
Adoption Trends: Data from 2020–2021 highlights an accelerated adoption of digital payments.
Targeted Expansion: Recommendations include focusing marketing and merchant onboarding in semi-urban and rural districts with high population density but lower current activity.
Infrastructure: Improving digital infrastructure in lower-performing regions is essential for promoting financial inclusivity.
